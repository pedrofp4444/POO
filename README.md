## Building

### Dependencies

- Gradle
- JUnit5
- Java Docs 

### Building

A release build is achieved with:

```bash
$ ./gradlew build
```

To build our tests:
```bash
$ ./gradlew test
```

To remove files generated by a build, run:

```bash
$ ./gradlew clean
```

## Running

To run our application: 
```bash
$ ./gradlew run --console=plain
```
